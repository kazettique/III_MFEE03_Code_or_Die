#### 關於git的commit訊息 (2019/04/10) 
##### Git推送的commit訊息，前面加上英文字母代號 (英文名字第一個字母)，方便辨認，例如：
1. W: 表格更新
2. V: navbar樣式修改
3. 以此類推
